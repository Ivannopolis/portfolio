J’ai réalisé un portfolio sous forme de site web afin de présenter mes projets. Même s’il existe des outils professionnels pour créer ce type de support, j’ai chois d’apprendre le HTML et le CSS afin de bénéficier d’une plus grande liberté et de mieux comprendre la structure d’un site. Vous trouverez ci-dessous un exemple de test que j’ai intégré dans le cadre d’un projet académique. Bien que ce prototype ait pu fonctionner, je n’ai pas réussi à y implémenter correctement des animations et une mise en page responsive, éléments que je considérais essentiels. J’ai donc finalement opté pour l’utilisation d’un template, tout en capitalisant sur l’expérience acquise.
