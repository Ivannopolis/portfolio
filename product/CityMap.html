CityMap Interactive

projet scolaire
2025

Dans le cadre d’un projet en équipe de 5, nous avons conçu en C, avec la librairie GTK4, une carte interactive inspirée de Google Maps. L’utilisateur pouvait consulter les informations des bâtiments, calculer le plus court chemin avec l’algorithme de Dijkstra, ou encore sélectionner le meilleur bar selon ses préférences. Nous avons également intégré un mode ludique, avec des mini-jeux permettant de “traverser la ville”. Le développement s’est appuyé sur un dépôt GitHub collaboratif et un environnement Linux, avec la mise en place de tests unitaires pour valider chaque module du programme

  • Équipe3 persones
  • Logiciels:Ltspice, Kicad, C
